## Visual Inspiration Sources

The best Excalidraw-style diagrams share these traits (studied from Anthropic docs, ByteByteGo, and top engineering blogs):

### Anthropic Documentation Style
- Warm off-white backgrounds (#FAFAF8)
- Colored left-border bars on cards (4-6px)
- Role badges as pills on the right margin
- Dashed curved arrows for return flows
- Step labels (STEP 1, STEP 2) on the left
- Green-tinted cards for resolution/completion
- Minimal icons, maximum text clarity

### ByteByteGo Style
- Dark backgrounds with vibrant accent colors
- Real product icons for recognizable services
- Grouped containers with dashed borders
- Side-by-side comparison panels (vs. layouts)
- Color-coded sections per system component
- Bottom callout with the key insight

### Engineering Blog Style
- Color-coded flows (one color per data path)
- Shared processing zones with translucent backgrounds
- Multi-path routing with divergent outputs
- Attribution labels on every connection
- Input/output columns (left → right flow)

Use these as pattern references, but always apply the excalidraw-svg design system (warm backgrounds, dark text, pastel cards) for consistent output quality.
